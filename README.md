# Admin Dashboard

Modern enterprise-level admin dashboard built with React, TypeScript, TailwindCSS, and shadcn/ui.

## Features

- Authentication
- Role-Based Access Control (RBAC)
- Protected Routes
- CRUD Operations
- Responsive Dashboard
- Dark Mode
- Charts & Analytics
- TanStack Query
- Zustand State Management
- React Hook Form + Zod
- Mock API Integration
- Modern SaaS UI

## Tech Stack

- React
- TypeScript
- Vite
- TailwindCSS
- shadcn/ui
- TanStack Query
- Zustand
- React Router DOM
- React Hook Form
- Zod
- Recharts

## Getting Started

```bash
npm install
npm run dev
```
